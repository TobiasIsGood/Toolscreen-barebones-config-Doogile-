# Toolscreen barebones config (Doogile)
A preset config file to setup the barebones (Doogile) projectors and overlay with Toolscreen

- Download the latest version [here](https://github.com/TobiasIsGood/Toolscreen-barebones-config-Doogile-/releases)
- [Eye measuring overlay download](https://github.com/TobiasIsGood/Toolscreen-barebones-config-Doogile-/blob/8edfc9bcc560e0647fc819bf8706759284f20020/overlay.png)

 # How to use
  - Unzip the folder, and put the new config.toml in your toolscreen folder. (C:/Users/<your username>/.config/toolscreen/)
  - Once the game is open, press CTRL+I to open the toolscreen config menu. Here, you can change sensitivity, resizing hotkeys, rebinds, etc.
